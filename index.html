Fake news deduction system

TruthGuard – A Flask-Based Adaptive Fake News Detection System

TruthGuard is a machine learning–based web application developed using Flask that detects whether a news article is real or fake. The system uses TF-IDF vectorization and the Passive Aggressive learning algorithm to analyze textual data and classify news content accurately. It provides a simple interface where users can input news text and instantly receive a prediction, helping to combat misinformation and improve information reliability.

Python-Based Intrusion Detection and Prevention System (IDS/IPS)

This project is a Python-based security tool designed to monitor network traffic and detect suspicious or malicious activities.
